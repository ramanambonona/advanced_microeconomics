Théorie de l'optimisation dynamique

Problème d'optimisation dynamique

L'optimisation dynamique consiste à maximiser une fonction objectif intertemporelle :

$$\max_{\{c_t, k_{t+1}\}_{t=0}^T} \sum_{t=0}^T \beta^t U(c_t)$$

Sous contrainte de ressources à chaque période :

$$g(c_t, k_t, k_{t+1}) = 0$$

Pour l'horizon infini, la somme va de $t=0$ à $\infty$.

Équation d'Euler

La condition d'optimalité intertemporelle est donnée par l'équation d'Euler :

$$U'(c_t) = \beta U'(c_{t+1}) F'(k_{t+1})$$

Cette équation relie la consommation entre deux périodes adjacentes et le produit marginal du capital.

Équation de Bellman :

$$V(k_t) = \max_{c_t} \left\{ U(c_t) + \beta V(k_{t+1}) \right\}$$

Méthodes de résolution

Programmation dynamique

Équation de Bellman et principe d'optimalité

$$V(k) = \max_c \{U(c) + \beta V(k')\}$$

Contrôle optimal

Principe du maximum de Pontryagin

$$\mathcal{H} = U(c) + \lambda g(c,k,k')$$

État stationnaire

Analyse de long terme pour horizon infini

$$k_{t+1} = k_t = k^*$$

Exemples Classiques

Modèle de Croissance de Ramsey

$$\max \sum_{t=0}^\infty \beta^t \frac{c_t^{1-\gamma}}{1-\gamma}$$
$$\text{s.c. } k_{t+1} = A k_t^\alpha + (1-\delta)k_t - c_t$$

Croissance optimale avec utilité CRRA

Modèle de Consommation

$$\max \sum_{t=0}^T \beta^t \log(c_t)$$
$$\text{s.c. } a_{t+1} = (1+r)a_t + y_t - c_t$$
